Отчет Богданов Евгений 324

1) Был выбрал вариант Б0 (5-й в группе, о - 16-я буква алфавита)
2) Управление игроком осуществляется с помощью клавиш WASD
3) Реализован лабиринт с одним выходом из него с помощью 11 комнат разного типа (все команты хранятся в директории maps)
4) Карта с комнатами так же хранится в директории maps и называется "mother_level.txt"
5) Игрок перемещается между комнатами (по одной комнате на весь экран)
6) При попадании в пропасть (пустоту) игрок умирает, выводит сообщение о смерти игрока
7) Игрок не проходит через стены
8) Игрок собирает сокровища (в данном случае монетки)
9) При выходе из лабиринта отображается сообщение "you win"

Из дополнительного

10) Реализован плавный переход "затухание" между уровнями
11) Реализована анимация передвижения игрока (во все 4 стороны и даже по диагонали)
12) Реализована анимация монеток 
13) Реализован счетчик монеток в правом верхнем углу
14) Реализован механизм "запоминания" уровней (при возвращении на предыдущий уровень собранные монетки отображаться не будут)
